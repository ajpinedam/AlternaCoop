# Markdown
    
    Creado por John Gruber en el 2004, Markdown es un lenguaje de enmaquetado utilizado para darle formato a documentos de texto plano, es el mas famoso del mundo.

    A diferencia de un editor de texto, en Markdown los cambios hechos al texto no se aprecian al instante presionando botones, dependiendo de la sintaxis que le apliques, se veran cambios en el documento.

    Por ejemplo, para resaltar un encabezado se pone un simbolo de numero (#) al comenzar y para poner negritas se rodea con **asteriscos**. 

    Se puede agregar formato de Markdown con un simple editor de texto o con un software dedicado a esto.

## Por que usar Markdown?

    Las razon es por las que la gente prefiere usar Markdown en vez de un editor que te muestra al instante lo que obtendras:

- Puede ser utilizado para todo: websites, documentos, notas, libros, presentaciones, mensajes de email y documentacion tecnica.

- Es portable, si no te gusta la aplicacion que estas usando, puedes migrar a otra sin ningun problema.

- Se puede usar en cualquier dispositivo o sistema operativo.

- Esta a prueba de obsolesencia, si la aplicacion que usas deja de funcionar, podras usar tus textos en cualquier editor de texto.

- Esta en todas partes. Sitios como Reddit y Github lo soportan, y muchos otras aplicaciones web o de escritorio.

## Sobre la marcha

    Markdown se aprende usandolo y lo mejor es que nisiquiera hay que descargar nada, hay muchos editores online. El mas recomendado es JotBird.

## Como funciona?

    Cuando escribes Markdown se oculta lo que pasa detras de escena, entonces se guarda en un archivo de extension .md. Despues se necesita una aplicacion con la capacidad de procesarlo, convirtiendolo en HTML. 

## Para que es bueno?

    Es rapido y bueno para tomar notas, crear contenido para un sitio web y producir documentos listos para imprimir.

    Se aprende rapido y una vez lo aprendes, puedes usarlo para todo. Desde escribir correos, crear contenido para la web hasta la lista de compras.

### Websites

    Markdown fue hecho para la web, por lo que no deberia ser sorpresa que hay muchas aplicaciones especificamente creadas para crear contenido web.

    Esta blot.im, crea una carpeta en tu computadora para arrastrar y soltar tus archivos Markdown y aparecen en tu pagina.

    Para los que tienen familiaridad con HTML, CSS y control de versiones deben probar Jekyll. Transforma los archivos Markdown en HTML. Tambien provee hosting gratis para paginas de Github.

    Ghost sirve para manejar un sistema de manejo de contenido para blogs.

### Documentos

    No es lo mejor del mundo como Word, pero es lo suficientemente bueno para crear documentos basicos como tareas y cartas. Se puede exportar a PDF o HTML. Ya en PDF puedes usarlo para lo que sea como imprimirlo, enviarlo por correo o subirlo a un website.

    Aplicaciones para escribir documentos de Markdown:
    **
- Mac: MacDOwn, iA Writer, o Marked 2.
- iOs/Android: iA Writer.
- Windows: ghostwriter o Markdown Monster.
- Linux: ReText o ghostwriter.
- Web: JotBird, Dillinger, o StackEdit.

### Notas

    Casi de cualquier forma que se mire es la sintaxis ideal para tomar notas. Desafortunadamente, Evernote y OneNote, que son las aplicaciones mas populares de notas no lo soportan, pero estan otras que si:

- Obsidian
- Simplenote
- Notable
- Bear
- Joplin

### Libros

    Con servicios como Leanpub, puedes convertir archivos escritos en Markdown en libros electrónicos (PDF, EPUB, MOBI). Luego puedes subir el PDF a plataformas como Kindle Direct Publishing para imprimir versiones físicas

### Presentaciones

    También se pueden crear presentaciones usando Markdown. Aunque requiere práctica, puede ser más rápido que usar programas como PowerPoint. Herramientas populares incluyen Remark, Cleaver y Marp, además de aplicaciones como Deckset o Hyperdeck en Mac.

### Email

    Existe una extensión llamada Markdown Here que permite escribir correos en Markdown y convertirlos automáticamente a HTML listo para enviar.

### Collaboration

    Algunas aplicaciones de trabajo en equipo usan partes de Markdown para dar formato al texto (negritas, cursivas, etc.). Ejemplos son Slack, Discord, Wiki.js y Mattermost.

### Documentacion

    Markdown es muy usado para documentación técnica. Herramientas como Read the Docs, MkDocs, Docusaurus, VuePress y Jekyll permiten crear sitios web de documentación a partir de archivos Markdown.

## Sabores de Markdown

    No todas las aplicaciones usan exactamente la misma versión de Markdown; estas variantes se llaman “flavors” (sabores). Por eso, es importante revisar la documentación de cada aplicación para saber qué funciones soporta. Se recomienda usar una aplicación con buen soporte de Markdown para mantener la compatibilidad entre diferentes herramientas.
